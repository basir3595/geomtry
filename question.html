What is DOM and what is the purpose of DOM?

The Document Object Model (DOM) aims to make it easier for Javascript to access HTML.The document object represents the whole html document.When html document is loaded in the browser, it becomes a document object. It is the root element that represents the html document. It has properties and methods. By the help of document object, we can add dynamic content to our web page.
